# How we quote

Every project is custom-scoped after a 30-min intro call. Pricing ranges below are guidance, not a menu. Real numbers go in a one-pager sent after the call.

# Initial 4-layer deploy (one-time)

Typical range: **A$4,500 - A$15,000** depending on:
- Number of skills in scope (3-5 standard, 6-10 larger)
- Number of n8n workflows (2-4 standard, 5-8 larger)
- Surface complexity (Telegram bot: low / Slack bot: medium / custom dashboard: high / voice with ElevenLabs+Twilio: high)
- Integrations beyond the common set (CRM, Sheets, Gmail, Outlook are standard; custom APIs or legacy systems push quote up)
- Size of Layer 1 brain seed (15-25 files standard, 40+ for complex businesses)

Delivered in 4-8 weeks. Fixed price, no hourly dance.

Includes:
- Workflow audit (1-2 days onsite or calls)
- Git repo + 15-25 context files written with the client
- 3-5 skills authored, versioned, tested
- 2-4 n8n workflows wired to the client's existing stack
- One primary surface (Telegram, Slack, SMS, web dashboard, or voice)
- Training + handover docs + admin access

# Brain + skills growth (monthly retainer)

Typical range: **A$800 - A$2,500 / month** depending on scope. Minimum 3-month commitment, month-to-month after that.

Includes:
- Monitoring + alerts + error recovery on every workflow
- 2-3 new skills authored per month
- Brain grooming sessions (we sit with the team and extend Layer 1)
- New n8n automations as priorities shift
- Direct channel (Telegram/Slack) for questions, same-day response

# Website builds (standalone)

Typical range: **A$2,000 - A$6,000** for a marketing site or client portal. More complex dashboards scoped individually. 14-day turnaround for standard builds.

# Discounts

- 10% off first retainer month for referrals from existing clients
- Bundle discount: Initial deploy + first 3 months retainer booked together = 5% off the combined total

# What we don't charge for

- First 30-min intro call
- Initial scope proposal after intro call
- Small tweaks to the live system in the first 30 days post-deploy
- Brain-file grooming in the first retainer month (this is expected to be heavy)

# Walk-away signals

- Budget under A$2,000 total (we're not the right fit; suggest Zapier + a ChatGPT subscription)
- Timeline under 2 weeks for an initial deploy (cannot do it well)
- Asking to pay hourly without a scoped build
