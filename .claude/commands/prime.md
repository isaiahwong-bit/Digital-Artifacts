- **Uncommitted:** any uncommitted changes under `brain/` or `skills/` (from `git status -s`).
- **Recent decisions:** the last 2 to 3 rows of the decision log.

Keep the report tight. Do not restate file contents. End by asking what we are working on if the
session has no task yet.
